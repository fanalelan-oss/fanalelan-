## 🎯 نظرة عامة على المشروع

يهدف هذا الموقع إلى أن يكون البوابة الرقمية لشركة "فن للإعلان"، حيث يعرض خدمات الشركة المتنوعة وأعمالها السابقة بطريقة احترافية وجذابة. تم تصميمه ليكون أداة تسويقية فعالة، مع التركيز على تحسين تجربة المستخدم وتسهيل عملية التواصل للعملاء المحتملين للحصول على استشارات وعروض أسعار.

## ✨ الميزات الرئيسية

- **تصميم عصري وداكن:** واجهة مستخدم أنيقة مع نظام ألوان داكن وذهبي يعكس فخامة وجودة الخدمات.
- **تجربة مستخدم تفاعلية:** استخدام تأثيرات حركية (Animations) مدروسة لجعل تجربة التصفح ممتعة وجذابة.
- **أداء فائق السرعة:** مبني باستخدام أحدث تقنيات Next.js لضمان تحميل سريع للصفحات.
- **متجاوب مع جميع الأجهزة (Fully Responsive):** تصميم يضمن عرضًا مثاليًا على كافة الشاشات، من الهواتف المحمولة إلى شاشات سطح المكتب الكبيرة.
- **مدونة متكاملة:** قسم للمدونة لعرض مقالات متخصصة، مما يساهم في إثراء المحتوى وتحسين محركات البحث (SEO).
- **صفحات خدمات ديناميكية:** لكل خدمة صفحة خاصة بها مع محتوى وصور مخصصة.
- **نموذج تواصل فعال:** نموذج اتصال آمن ومباشر باستخدام React Email و Resend.
- **تحسين محركات البحث (SEO) متقدم:**
  - إنشاء تلقائي لخريطة الموقع (`sitemap.xml`) و ملف `robots.txt`.
  - بيانات وصفية (Metadata) غنية وديناميكية لكل صفحة.

## 📂 هيكلية المشروع

تم تنظيم المشروع ليكون سهل الفهم والصيانة. إليك نظرة على أهم المجلدات:

```
fanalelan-/
├── public/                  # يحتوي على الأصول الثابتة (صور، أيقونات)
│   ├── blog_images/
│   └── images/
├── src/
│   ├── app/                 # يحتوي على صفحات الموقع (App Router)
│   │   ├── (main)/          # الصفحات الرئيسية
│   │   ├── blog/            # صفحات المدونة
│   │   ├── services/        # صفحات الخدمات
│   │   └── sitemap.ts       # ملف إنشاء خريطة الموقع الديناميكي
│   ├── components/          # مكونات React القابلة لإعادة الاستخدام
│   │   ├── ui/              # مكونات واجهة المستخدم الأساسية
│   │   └── email/           # قالب البريد الإلكتروني
│   └── lib/                 # يحتوي على البيانات والوظائف المساعدة
│       ├── blog-data.ts     # بيانات ومحتوى مقالات المدونة
│       ├── data.ts          # بيانات الخدمات والمشاريع وغيرها
│       └── types.ts         # تعريفات TypeScript
├── .env.local.example       # ملف مثال لمتغيرات البيئة
├── next.config.mjs          # إعدادات Next.js
└── README.md                # هذا الملف
```

## 🚀 التشغيل والتطوير المحلي

لتشغيل نسخة من المشروع على جهازك المحلي، اتبع الخطوات التالية:

1.  **استنساخ المستودع:**
    ```bash
    git clone https://github.com/fanalelan-oss/fanalelan-.git
    cd fanalelan-
    ```

2.  **تثبيت الحزم:**
    ```bash
    npm install
    ```

3.  **إعداد متغيرات البيئة:**
    - قم بنسخ ملف `.env.local.example` إلى ملف جديد باسم `.env.local`.
    - أضف مفتاح API الخاص بك من [Resend](https://resend.com/) لإتاحة خاصية إرسال رسائل البريد الإلكتروني من نموذج التواصل.
    ```
    RESEND_API_KEY="YOUR_API_KEY"
    ```

4.  **تشغيل خادم التطوير:**
    ```bash
    npm run dev
    ```

5.  افتح [http://localhost:3000](http://localhost:3000) في متصفحك لترى الموقع.

## ☁️ النشر (Deployment)

يتم نشر هذا المشروع باستخدام **Firebase App Hosting**، والذي يتكامل مباشرة مع GitHub. أي عملية `push` إلى الفرع `main` ستقوم تلقائيًا بتشغيل عملية بناء (build) ونشر (deploy) نسخة جديدة من الموقع، مما يضمن أن الموقع المباشر محدث دائمًا بأحدث التغييرات.
